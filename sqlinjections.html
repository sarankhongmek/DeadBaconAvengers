SQL injections

SQL (Structured Query Language) injection is a set of SQL commands that is placed in URL string or in any places where there is inputs that require users input to perform different type of queries that is not allowed,To a retrieve a responses from the website server that is connected with the client side application and manipulate data in the data base.

This attack works when the web application does not have any user input validation in place properly before passing them to SQL statement. Some of common places injections is perform is in address bar, search field and data fields.

One easy way to test if a web application has an SQL exploit is to use “ ‘ “ string character in any input fields and see if you get any error.
